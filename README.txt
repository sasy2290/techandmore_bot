# TechAndMore Bot (Amazon PA-API)
1. Inserisci i segreti su Replit:
   - BOT_TOKEN
   - CHANNEL_ID
   - POST_INTERVAL_HOURS = 0.25
   - MIN_DISCOUNT = 25
   - PAAPI_ACCESS_KEY
   - PAAPI_SECRET_KEY
   - PAAPI_HOST = webservices.amazon.it
   - PAAPI_REGION = eu-west-1
   - PARTNER_TAG = techandmore05-21
   - PARTNER_TYPE = Associates

2. Installa dipendenze:
   pip install -r requirements.txt

3. Esegui:
   python bot.py

4. Copia URL .repl.co e monitora con UptimeRobot.
